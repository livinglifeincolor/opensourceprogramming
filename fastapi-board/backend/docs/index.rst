FastAPI Board Documentation
============================

Welcome to the technical documentation for **FastAPI Board** — a minimal
bulletin-board REST API built with FastAPI, asyncpg, and PostgreSQL.

The documentation is automatically generated from Python docstrings using
`Sphinx <https://www.sphinx-doc.org/>`_ with the
`autodoc <https://www.sphinx-doc.org/en/master/usage/extensions/autodoc.html>`_
extension.  All docstrings follow the
`Google style guide <https://google.github.io/styleguide/pyguide.html#38-comments-and-docstrings>`_.
